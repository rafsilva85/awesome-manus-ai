# Contributing to Awesome Manus AI

Thank you for your interest in contributing! This list is community-driven and we welcome all contributions.

## How to Contribute

1. Fork this repository
2. Add your resource to the appropriate section in `README.md`
3. Follow the format: `- [Resource Name](URL) - Brief description`
4. Submit a Pull Request

## Guidelines

- Make sure the resource is related to Manus AI
- Check that the link is working
- Write a clear, concise description
- Place the resource in the correct category
- One resource per Pull Request is preferred

## Adding a New Category

If your resource doesn't fit any existing category, feel free to suggest a new one in your PR description.

## Quality Standards

- Resources should be actively maintained
- No spam or self-promotion without genuine value
- Prefer open-source resources when possible
- Include pricing information for paid resources

## Code of Conduct

Be respectful and constructive. We're all here to help the Manus AI community grow.
